<!--more-->

有两种方法

1. 使用flex

   父元素：

   ```css
       width: 100vw;
       height: 500px;
       background: skyblue;
       /* position: relative; */
       display: flex;
       /* align-content: center; */
       justify-content: center;
       align-items: center;
   ```

   子元素不用设置

2. 使用transform

   父元素：

   ```css
       width: 100vw;
       height: 500px;
       background: skyblue;
       position: relative;
   ```

   子元素：

   ```css
       position: absolute;
       top: 50%;
       left: 50%;
       width: 200px;
       transform: translate(-50%, -50%);
       word-break: break-all;
       background-color: deepskyblue;
   ```

引用值即对象，使用new后跟构造函数创建，不同于原始值，引用值保存在堆空间中。

<!--more-->

# String

```js
s1 = "text"
s2 = s1.substring(2)

//等同于
s1 = new String("text")
s2 = s1.substring(2)
s1 = null;
```

方法

```js
str = "hello"
s2 =  "world"
str.concat(s2)//"helloworld"
//提取子串，两个参数，子串开始和结束位置，左闭右开
str.slice(2)//  "llo"
str.substring(2)//llo
str.substr//第二个参数是长度
str.slice(-1)//o

```



# Number

方法

```js
toString(a) //返回以a为基数的字符串
"10".toString(2)//"1010"

toFixed(a)//返回包含a位小数的字符串
10.toFixed(2)//"10.00"

```

Math.max()
